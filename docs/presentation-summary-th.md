# สรุปสำหรับพรีเซนต์: Humanoid Robot Reinforcement Learning

## ภาพรวมโปรเจกต์
โปรเจกต์นี้มุ่งสอนหุ่นยนต์ humanoid รุ่น Unitree G1 ให้เรียนรู้การหยิบจับและวางวัตถุด้วย Reinforcement Learning ในสภาพแวดล้อมจำลอง ManiSkill3 โดยใช้อัลกอริทึม PPO เป็นแกนหลักของการฝึก

## เป้าหมายของงาน
- ให้หุ่นยนต์เรียนรู้การทำงานเองจากการลองผิดลองถูก แทนการเขียนลำดับคำสั่งแบบตายตัว
- ทดสอบความสามารถของ humanoid robot ในงาน manipulation ที่มีหลายขั้นตอน
- ออกแบบ reward function ให้ช่วยให้การเรียนรู้เสถียรและมีประสิทธิภาพ

## งานที่ทำในโปรเจกต์
### Task A: วางแอปเปิ้ลลงในชาม
- มือขวาหยิบแอปเปิ้ล
- เคลื่อนแอปเปิ้ลไปเหนือชาม
- ปล่อยให้แอปเปิ้ลลงในชามอย่างถูกต้อง

### Task B: ย้ายชามไปยังอ่างล้างจาน
- มือซ้ายหยิบชามที่มีแอปเปิ้ลอยู่ข้างใน
- เคลื่อนชามไปยัง sink
- วางชามลงโดยไม่ให้แอปเปิ้ลหล่น

## วิธีการที่ใช้
### 1. Reinforcement Learning ด้วย PPO
- PPO เป็นอัลกอริทึม RL ที่มีความเสถียรและนิยมใช้อย่างแพร่หลาย
- โมเดลแบ่งเป็น 2 ส่วน:
  - Actor ทำหน้าที่ตัดสินใจว่าจะขยับข้อต่ออย่างไร
  - Critic ทำหน้าที่ประเมินว่าสถานะปัจจุบันดีแค่ไหน
- จุดเด่นของ PPO คือมี clipping ช่วยไม่ให้ policy เปลี่ยนเร็วเกินไปในแต่ละรอบ

### 2. Reward Function แบบเป็นขั้นตอน
- ออกแบบ reward ให้สอดคล้องกับลำดับการทำงานจริง เช่น เข้าใกล้วัตถุ จับวัตถุ ย้ายวัตถุ และวางสำเร็จ
- วิธีนี้ช่วยลดปัญหา sparse reward และทำให้หุ่นยนต์เรียนรู้ได้ง่ายขึ้น

### 3. การฝึกแบบขนานบน GPU
- จำลอง environment พร้อมกัน 1024 ตัว
- ช่วยเก็บประสบการณ์ได้เร็วและลดเวลาการ train อย่างมาก
- รองรับการฝึกระยะยาวที่ระดับ 200 ล้าน timesteps

## ปัจจัยสำคัญที่ทำให้โปรเจกต์สำเร็จ
- Reward shaping ที่ออกแบบเป็นลำดับขั้นชัดเจน
- การใช้ GPU simulation แบบขนานจำนวนมาก
- การเลือก observation ที่จำเป็นต่อการตัดสินใจ เช่น ตำแหน่งวัตถุ ตำแหน่งมือ และสถานะการจับ
- การตั้งค่า simulation ให้เหมาะกับทั้งความเร็วและความเสถียร

## ผลลัพธ์
- Task A ทำได้สำเร็จ 100% ที่ checkpoint 1251
- ค่า eval return ของ Task A เท่ากับ 155.43
- Task B ยังอยู่ระหว่างการพัฒนา เพราะเป็นงานที่ยากกว่าและต้องควบคุมไม่ให้แอปเปิ้ลหล่นจากชาม

## Tech Stack
- Simulator: ManiSkill3 + SAPIEN
- Physics Engine: PhysX 5 (CUDA)
- Deep Learning: PyTorch
- RL Algorithm: PPO
- Robot Model: Unitree G1

## ประเด็นที่ควรเน้นตอนพรีเซนต์
1. โปรเจกต์นี้ไม่ได้สั่งหุ่นยนต์แบบทีละขั้น แต่ให้มันเรียนรู้เองจาก reward
2. หัวใจของความสำเร็จคือการออกแบบ staged reward ที่ดี
3. การใช้ GPU จำลองหลาย environment พร้อมกันทำให้การฝึกเป็นไปได้จริง
4. ผลลัพธ์ของ Task A แสดงให้เห็นว่าแนวทางนี้ใช้ได้ผลกับงาน manipulation ที่มีหลายขั้นตอน
5. Task B เป็นก้าวต่อไปที่ท้าทายกว่า เพราะต้องย้ายวัตถุโดยรักษาสถานะของสิ่งที่อยู่ภายในไปพร้อมกัน

## สคริปต์สั้นสำหรับพูดพรีเซนต์
โปรเจกต์นี้เป็นการสอนหุ่นยนต์ humanoid Unitree G1 ให้เรียนรู้การหยิบและวางวัตถุด้วย Reinforcement Learning ใน simulation โดยเราใช้ PPO เป็นอัลกอริทึมหลัก และใช้ ManiSkill3 เป็น environment สำหรับฝึก

หัวใจสำคัญของงานนี้คือการออกแบบ reward function แบบ staged reward ซึ่งแบ่งการให้คะแนนตามลำดับขั้นของงานจริง เช่น เข้าใกล้วัตถุ จับวัตถุ เคลื่อนย้าย และวางสำเร็จ ทำให้หุ่นยนต์เรียนรู้ได้เร็วและเสถียรมากขึ้น

ผลลัพธ์ที่ได้คือ Task A ซึ่งเป็นการหยิบแอปเปิ้ลไปวางในชาม สามารถทำได้สำเร็จ 100% ส่วน Task B ที่ต้องย้ายชามทั้งใบไปยังอ่างล้างจานยังอยู่ระหว่างการพัฒนา แต่ก็สะท้อนให้เห็นถึงทิศทางที่ดีของระบบนี้ในการต่อยอดไปสู่งานที่ซับซ้อนขึ้น
